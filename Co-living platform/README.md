# 🌿 COKKIRI: Co-living Platform
**Smart Living, Togetherness Redefined. © 2026**

이 프로젝트는 프리미엄 에디토리얼 무드의 **'Moss & Aloe'** 디자인 시스템을 기반으로 구축된 스마트 코리빙 플랫폼입니다. IoT 기반의 보완 관리, 커뮤니티 공간 예약, 그리고 세련된 주거 경험을 제공합니다.

---

## 👥 팀원 가이드 (Full-Stack Team of 6)

모든 팀원은 **UI_GUIDELINE.md**를 필독한 후 개발을 진행해 주세요. AI를 활용해 코드를 짤 때 해당 가이드를 컨텍스트로 제공하면 일관성 있는 UI를 얻을 수 있습니다.

- **Frontend**: Vite + HTML + Javascript + Tailwind CSS v4
- **Styling**: `src/styles/theme.css`가 전역 테마 및 변수의 중심입니다.
- **Components**: `src/app/components/`의 공통 컴포넌트를 사용하여 재사용성을 높여주세요.

## 🚀 시작하기

1.  **의존성 설치**:
    ```bash
    npm install
    ```

2.  **로컬 서버 실행**:
    ```bash
    npm run dev
    ```

3.  **디자인 시스템 확인**:
    - [UI_GUIDELINE.md](./UI_GUIDELINE.md)를 통해 색상, 서체, 간격 규칙을 확인하세요.
